# Mangas VXL — Proyecto Final Diseño Web

Tienda de comercio electrónico de mangas en español, desarrollada como proyecto integrador del curso Diseño Web.

## Descripción del negocio

**Mangas VXL** es una tienda en línea especializada en la venta de mangas en español. Ofrecemos los títulos más populares de géneros como Shonen, Accióny Fantasía, con una experiencia de compra rápida y moderna.
